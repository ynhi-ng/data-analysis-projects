# World Development Indicators — EDA Report

## Summary
Exploratory data analysis of world development indicators,
examining trends and relationships across countries and regions.

## Tools Used
- R
- R Markdown
- Libraries: ggplot2, dplyr, tidyverse

## Files
- `world_development.Rmd` — main R Markdown analysis file
- `WorldBankGroup_FDI.csv` — FDI dataset sourced from World Bank (public)

## Data Sources
- WorldBankGroup FDI data: downloaded from World Bank Open Data
- World Development Indicators
